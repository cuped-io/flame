cuped.io flame v0.2.0 — script-tag change-type test page

Each block below is a target for one of the 8 change types flame supports as of v0.2.0. Configure variants in your cuped dashboard targeting these selectors, then load this page with the script tag pointed at your DSN.

1. text — replace textContent

Original text content. (selector: #target-text) 

{ "type": "text", "selector": "#target-text", "value": "New text!" }

2. html — replace innerHTML

Original markup. (selector: #target-html) 
{ "type": "html", "selector": "#target-html", "value": "<strong>New markup</strong>" }

3. attribute — set arbitrary attribute

Original href: https://example.com/original 
{ "type": "attribute", "selector": "#target-attribute", "attribute": "href", "value": "https://example.com/new" }

4. class — add/remove classes

classes: keep-this remove-me. (selector: #target-class) 

{ "type": "class", "selector": "#target-class", "add": ["new-class"], "remove": ["remove-me"] }

5. style — inline styles

Default color. (selector: #target-style) 

{ "type": "style", "selector": "#target-style", "styles": { "color": "#10b981", "font-weight": "700" } }

6. css — inject <style> tag globally

Restyled by global CSS rule. (class: .target-css) 

{ "type": "css", "css": ".target-css { background: #fef3c7; border-style: solid; }" }

7. visibility — show/hide

Visible. (selector: #target-visibility) 
{ "type": "visibility", "selector": "#target-visibility", "visible": false }

8. redirect — force navigation

Cannot show inline — when active, this variant immediately navigates the page elsewhere. 

{ "type": "redirect", "url": "/some-other-page" }

Controls

After loading: open DevTools → Console for the [Flame] debug logs (if data-debug="true" is set on the script tag), and Network for the API calls (/experiments/active, /assign, /observations).
